# Assets Policy

This repository keeps third-party datasets and downloaded media out of Git by default.

Safe-to-commit frontend demo assets live under
`frontend/public/assets/demo/`. They must be original project assets, generated
assets, or media with a clear license that permits redistribution in this
open-source repository.

Do not commit downloaded datasets such as the UCSD/WSDOT traffic database. Keep those
files local under `assets/` or `data/`, and document their source instead.
